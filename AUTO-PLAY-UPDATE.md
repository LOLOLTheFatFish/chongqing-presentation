### **场景1: 完整自动播放**
1. **点击"🎯 Auto Play Presentation"**
2. **自动加载第一页** + **1秒后自动播放音频**
3. **音频结束** → **自动跳转到下一页** → **自动播放下一页音频**
4. **循环播放**直到最后一页
5. **最后页播放完毕** → **停止播放**

### **场景2: 手动控制模式**
1. **点击"🎯 Auto Play Presentation"**
2. **点击播放按钮**手动开始
3. **使用"下一页"按钮**手动翻页
4. **音频播放完毕**不会自动翻页 (手动模式)

### **场景3: 暂停和恢复**
1. **播放过程中** → **点击暂停** → **音频和自动翻页停止**
2. **再次点击播放** → **继续自动播放流程**

## 🛠️ **技术实现**

### **自动播放逻辑**
```javascript
// 点击开始按钮
startPresentation(slideIndex) {
    isAutoPlayEnabled = true;  // 启用自动播放
    updateSlide();              // 加载当前页面
    loadCurrentAudio();         // 加载音频文件
    
    // 1秒后自动开始播放
    setTimeout(() => {
        if (isAutoPlayEnabled) {
            toggleAudio();       // 开始播放
        }
    }, 1000);
}

// 音频结束处理
onAudioEnded() {
    if (isAutoPlayEnabled && currentSlideIndex < slides.length - 1) {
        // 自动跳转到下一页
        nextSlide();
        // 0.5秒后自动播放下一页音频
        setTimeout(() => {
            if (isAutoPlayEnabled) {
                toggleAudio();
            }
        }, 500);
    }
}
```

- **自动隐藏**: 3秒无操作后自动隐藏
